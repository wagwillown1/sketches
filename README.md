# Sketches

A minimal two-page static site for drafting and testing ideas.

## Run locally

1. Clone or download this repository.
2. Open `index.html` in your browser (no build step required).

## Deploy to GitHub Pages

1. Push to the `main` branch.
2. In GitHub, go to **Settings â†’ Pages**.
3. Select the `main` branch and the **root** folder, then save.
4. Your site will be available at the GitHub Pages URL (works from the `/sketches` subpath thanks to relative links).
